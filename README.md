# 考评系统
```
vue+django前后端分离，strive2web中为后端程序文件，其余为前端。
```
## 前言
受单位领导要求，为一项考评试点任务开发在线考评及后台管理系统，首次进行web开发，两个月边学边做，形成了这个简单的XXX考评系统。
>前端vue、后端django、数据库sqlite、服务器apache

# 前端开发
需要安装npm
## vue前端初始化
```
npm install
```

### 启动vue热编译开发服务
```
npm run serve
```

### vue打包和压缩
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
